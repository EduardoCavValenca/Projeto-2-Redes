## Como rodar o código
Para compilar o programa, digite no terminal:
```
make
```
Para executar o programa, digite:
```
make run
```
Para apagar os arquivos criados pelo comando ```make```, digite:
```
make clean
```

## Requisitos
g++:
```
sudo apt install g++
```
